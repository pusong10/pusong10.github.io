开发者概览 (About Me)

C# 开发工程师 | 工业通讯与高性能渲染专家

专注于构建稳定、高效的工业级上位机系统。具备从底层驱动封装到高频数据实时可视化的全栈开发经验。

🛠 核心技术栈

  • 架构方案: 深耕 WPF 开发，精通 MVVM 模式及 Prism 模块化框架应用 。
  • 高性能渲染: 具备 SkiaSharp 深度开发经验，主导过工业级 十万级数据点 实时动态波形渲染优化 。
  • 工业通讯: 熟练掌握 SerialPort (RS232/485)、TCP/UDP Socket 通讯；熟悉 Modbus RTU/TCP 协议及自定义二进制协议封装 。
  • 并发控制: 熟练处理高频采样下的 UI 异步更新；利用 System.Threading.Channels 构建“接收、解析、存储、渲染”四级解耦流水线 。

🚀 项目精华

高性能上位机通讯与控制系统

  • 架构优化: 构建基于 Task 的多线程并发模型，利用 Channels 机制规避 I/O 阻塞对 UI 响应的影响，确保系统长效运行稳定性 。
  • 稳定性维护: 优化物理层丢弃策略与自定义超时检测算法，将链路异常后的自动恢复时间缩短至 秒级 。
  • 流式解析: 利用 有限状态机 (FSM) 与 CRC16 校验算法精准提取 Payload，配合环形缓冲区解决粘包与断帧难题。

多协议工业网关与自动化配置平台

  • 驱动封装: 支持通过 XML/JSON 配置文件动态加载设备点表，实现异构设备（如西门子、欧姆龙等）数据的统一采集。
  • 调度算法: 设计基于 优先级队列 (Priority Queue) 的动态轮询策略，解决上百台从站设备并发通讯时的带宽竞争与延时波动。
